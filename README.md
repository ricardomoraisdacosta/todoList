Running Instructions

The Backend service will run in port : 3002
the Frontend service will run in port : 8888

Via NPM:

You should Run

For The Backend : In the root of the Project

### `cd Backend`

### `yarn`

### `yarn start`

or

### `cd Backend`

### `npm install`

### `npm start`

For the Frontend : In the root of the Project

### `cd Frontend`

### `yarn`

### `yarn start`

or

### `cd Frontend`

### `npm install`

### `npm start`
